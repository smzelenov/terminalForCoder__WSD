## Contributing

*1 - Fork it!*

*2 - Clone*

*3 - Create your feature branch:*
```bash
$ git checkout -b my-new-feature origin/master
```
*4 - Commit your changes:*
```bash
$ git commit -m 'Add some feature'
```
*5 - Push to the branch:*
```bash
$ git push origin my-new-feature
```
*6 - Submit a pull request*
